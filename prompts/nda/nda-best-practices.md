# Промпт: NDA — соответствие лучшим практикам

## Описание  
Промпт для проверки вашего договора NDA на соответствие рекомендуемым юридическим стандартам (структура, ответственность, исключения, механизм разрешения споров).

## Пример промпта для ИИ  

Ты — юрист по международному праву.
Проверь мой NDA на соответствие лучшим практикам:

Достаточно ли чётко сформулированы все ключевые условия?

Все ли обязательные элементы договора включены?

Есть ли пункты по исключениям, процедурам раскрытия данных третьим лицам, сроку действия?

Соответствует ли договор действующему законодательству (указать страну)?

[Вставьте текст]
