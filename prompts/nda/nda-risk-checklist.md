# Промпт: Анализ рисков NDA

## Описание  
Промпт для выявления типовых рисков и недочетов в договоре о неразглашении (NDA). Особенно полезен для быстрой проверки шаблонных документов.

## Пример задачи  
Проверьте следующий текст договора NDA на наличие:
- Пробелов в определениях сторон и защищаемой информации
- Недостаточно конкретных условий ответственности
- Проблем с сроком действия
- Отсутствия механизма разрешения споров

## Пример промпта для ИИ  
Ты — опытный юрист по контрактному праву РФ.
Проанализируй представленный текст договора NDA на наличие типовых рисков.
Для каждого найденного риска опиши:

Суть проблемы (например: отсутствие условия о сроке действия)

Возможные последствия для сторон (например: нельзя взыскать убытки)

Как рекомендуется устранить (например: добавить пункт о сроке и продлении)

Текст договора:

[Вставьте текст NDA]


## Ожидаемый результат  
- Структурированный список рисков
- Краткое пояснение последствий
- Конкретные рекомендации по доработке

## Комментарии  
- Можно применять для любых NDA, адаптируя промпт под соответствующее законодательство.
- Чем подробнее исходный текст, тем точнее анализ.
