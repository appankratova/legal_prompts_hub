# Проверка и анализ NDA

В этой папке собраны промпты для проверки типовых договоров NDA (соглашений о неразглашении информации).
- Каждый промпт предназначен для анализа юридических рисков, поиска слабых мест и автоматизации рутинных этапов работы.
- Используйте шаблон для создания новых промптов или предложите свои решения через Pull Request.

Рекомендуемые файлы:
- nda-risk-checklist.md — базовый чек-лист для анализа рисков
- nda-best-practices.md — шаблон для проверки на соответствие лучшим практикам
