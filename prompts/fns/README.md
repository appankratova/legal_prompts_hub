# Проверка контрагентов по данным ФНС

В этой папке собраны промпты и кейсы для автоматизированной проверки контрагентов по данным ФНС России (ЕГРЮЛ/ЕГРИП) через API‑ФНС и другие открытые источники.  
Кейсы предназначены для due diligence, KYC, M&A‑проверок и базовой оценки благонадёжности контрагентов.

## Как это работает

1. Технический скрипт (пример: `prompts/scripts/fns_test.py`) делает запрос к API‑ФНС по ИНН.  
2. Скрипт выбирает одно **действующее** юридическое лицо и возвращает только:
   - полное наименование организации (`НаимПолнЮЛ`),
   - ИНН,
   - статус,
   - полный адрес (`АдресПолн`).
3. Полученные данные передаются в LLM по одному из промптов из этой папки для юридического анализа.

Подробный техно-гайд по интеграции с госреестрами (ФНС, ЕГРЮЛ, КАД Арбитр и др.) см. в `docs/gov_registry_integration.md`.

## Связанные материалы
