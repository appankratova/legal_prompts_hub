# Анализ и автоматизация M&A (слияния и поглощения)

В этой папке собраны промпты для:
- анализа проектов M&A-договоров,
- проверки и дью-дилиженса целевых компаний,
- автоматизированной подготовки disclosure letter,
- оценки судебных рисков по сделке.

**Интеграции с государственными реестрами и сервисами:**
- [ЕГРЮЛ (единый госреестр юр. лиц)](https://egrul.nalog.ru) — реквизиты, собственники, история изменений, признаки недобросовестности
- [Росреестр](https://rosreestr.gov.ru) — реестр прав на недвижимость и обременений
- [КАД Арбитр](https://kad.arbitr.ru) — вся судебная практика компаний (истец/ответчик)
- [СПАРК/Контур.Фокус](https://spark-interfax.ru) — аналитика, связи и расширенные данные для due diligence

**Автоматизация:**
- Можно использовать готовые и открытые API Гос.сервисов (например, [API ФНС](https://api-fns.ru/api_help)) для комплексного сбора и проверки данных.
