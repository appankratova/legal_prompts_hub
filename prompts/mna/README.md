# Анализ договоров M&A (слияния и поглощения)

Здесь собраны промпты для проверки, анализа и автоматизации юридических задач по сделкам M&A.
- Промпты помогут выявлять риски, проверять структуру сделки, анализировать права и обязанности сторон.
- Основаны на лучших практиках международного и российского права.
