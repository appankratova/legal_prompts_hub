# Промпт: Проверка рисков M&A-договора

## Описание  
Промпт предназначен для выявления ключевых юридических рисков, типовых ошибок и “узких мест” в соглашениях о слиянии и поглощении компаний.

## Пример задачи  
- Найти отсутствующие или нечетко прописанные условия (предмет сделки, корпоративные права, гарантии и заверения, механизмы Price Adjustment)
- Оценить качество дью-дилиженс
- Выделить нетипичные риски

## Пример промпта для LLM  
Ты — эксперт по сделкам M&A (Россия и международное право).
Проанализируй следующий текст соглашения на наличие юридических рисков. Для каждого риска:

Опиши суть проблемы (например: отсутствует пункт о действии force majeure)

Объясни последствия (как это повлияет на стороны, возможность судебного иска)

Дай рекомендации по исправлению

Текст договора:

[Вставьте основной текст сделки]


## Ожидаемый результат  
- Список рисков по категориям (корпоративные, финансовые, процедурные, налоговые, репутационные)
- Краткое пояснение и рекомендации по доработке

## Комментарии  
- Полезно использовать для проверки драфта до переговоров и заключения сделки
- Применимо как для российских, так и для международных M&A
