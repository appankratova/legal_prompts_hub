# Промпт: Автоматизация подготовки Disclosure Letter для M&A

## Описание  
Промпт для автоматизированной генерации Disclosure Letter: выявление существенных фактов, составление списка раскрываемых обстоятельств, структурирование информации для покупателя.

## Пример задачи  
- Сформировать перечень исключений к гарантиям и заверениям
- Создать структурированный Disclosure Letter из входных документов и данных реестров

## Пример промпта для LLM  
Ты — корпоративный юрист, готовишь Disclosure Letter для сделки M&A.
На основе текста договора, результат

Составь таблицу существенных раскрываемых обстоятельств (сущность, дата, источник)

Раздели исключения по категориям (корпоративные, налоговые, судебные, IP, недвижимость)

Оцени полноту Disclosure Letter по практике российского и международного права

Договор/Term Sheet: [текст]

Due Diligence Report: [резюме]

Реестры: [данные по ЮЛ, недвижимости, судебным спорам и пр.]


## Ожидаемый результат  
- Таблица исключений с пояснением
- Оценка рисков по нераскрытым обстоятельствам
- Рекомендации по доработке

## Комментарии  
- Можно интегрировать парсер для получения данных из ЕГРЮЛ, Роспатента, Росреестра, КАД Арбитр
- Промпт особенно полезен в англо- и русскоязычных M&A
