# Промпт: Анализ судебных перспектив сделки M&A

## Описание  
Промпт для оценки вероятности успешного судебного спора по условиям сделки M&A: выделение типовых позиций практики, рисков по расторжению, доказывания нарушения заверений.

## Пример задачи  
- Комментировать, какие пункты чаще вызывают споры
- Оценить доказательственную базу для защиты интересов сторон

## Пример промпта для LLM  
Ты — юрист по судебным спорам M&A.
Проанализируй текст договора и собранные данные (в том числе из судебных реестров):

Каковы перспективы защиты интересов покупателя/продавца по спорным пунктам договора?

Какие прецеденты из базы КАД Арбитр наиболее релевантны?

Дай рекомендации по изменению положений договора для минимизации судебных рисков.
Текст договора: [приложи]
Данные: [выжимка из КАД, судебная практика по расторжению M&A]


## Ожидаемый результат  
- Структурированное мнение по каждому спорному пункту
- Ссылки на релевантные судебные дела
- Практические рекомендации по драфтингу

## Комментарии  
- Для расширенной автоматизации — интеграция с API КАД Арбитр, СПАРК, Росреестр
