# Промпт: Сравнение решений ФАС по типовым основаниям

## Описание  
Промпт для поиска закономерностей рассмотрения схожих споров: какие аргументы чаще приводят к успеху, какие типы нарушений признаются судом.

## Пример промпта  
Ты — эксперт по антимонопольным спорам.
Проведи сравнительный анализ двух решений ФАС по одному основанию (например, злоупотребление доминирующим положением).
Выдели:

Общие и отличительные черты аргументов сторон

Почему решения были разными (или одинаковыми)

Практические выводы для юриста по будущим делам

Решения:

[Вставьте текст двух решений]


## Ожидаемый результат  
- Сравнительная таблица аргументов
- Краткие практические рекомендации
