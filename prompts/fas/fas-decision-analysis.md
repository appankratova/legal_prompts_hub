# Промпт: Анализ решения ФАС

## Описание  
Промпт для автоматизированного выделения ключевых аргументов сторон, существа антимонопольного нарушения, логики принятия решения и рекомендаций для юристов по аналогичным кейсам.

## Пример задачи  
- Выделить аргументы истца и ответчика
- Сформировать список нормативных актов, упоминаемых в решении
- Предложить резюме для быстрой оценки вероятности успеха апелляции

## Пример промпта для LLM  

Ты — аналитик антимонопольной практики.
Дано решение ФАС по спору между компаниями.
Проанализируй текст решения и выдели:

Аргументы сторон (отдельно истец, отдельно ответчик)

Аргументы ФАС — почему принято такое решение

Нормативные акты, упоминаемые в решении

Советы для юриста по аналогичным спорам (меры, стратегия)
Текст решения:
[Вставьте решение ФАС]


## Ожидаемый результат  
- Чёткая классификация аргументов
- Таблица нормативных актов
- Краткое резюме для практического применения

## Комментарии  
- Промпт легко адаптируется для других ведомственных решений (ФНС, арбитраж)
- Если решение длинное — разбейте анализ на этапы (цепочка рассуждений, chain-of-thought)
