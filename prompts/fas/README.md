# Анализ решений ФАС РФ

В этой папке собраны промпты для исследования, автоматизированного анализа и кластеризации решений Федеральной антимонопольной службы.
- Используйте промпты для поиска типовых аргументов, выявления закономерностей рассмотрения споров, поиска ошибок или противоречий.
- Приглашаем делиться новыми шаблонами и кейсами через Pull Request.
