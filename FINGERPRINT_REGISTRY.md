# Реестр Fingerprints (FINGERPRINT_REGISTRY.md)

Уникальные идентификаторы промптов Legal Prompt Hub для защиты авторства и отслеживания плагиата.

## Формат

`LPH-YYYY-CAT-XXXX`

| Компонент | Описание |
|-----------|----------|
| `LPH` | Legal Prompt Hub |
| `YYYY` | Год создания |
| `CAT` | Категория (TPL, FNS, FAS, MNA, NDA, SCR) |
| `XXXX` | Порядковый номер (0001-9999) |

## Категории

| Код | Папка | Описание |
|-----|-------|----------|
| `TPL` | `/templates/` | Универсальные шаблоны |
| `FNS` | `/fns/` | Налоговое право |
| `FAS` | `/fas/` | Антимонопольное право |
| `MNA` | `/mna/` | M&A / Сделки |
| `NDA` | `/nda/` | Конфиденциальность |
| `SCR` | `/scripts/` | Скрипты автоматизации |

## Реестр

| Fingerprint | Название | Автор | Дата | Файл |
|-------------|----------|-------|------|------|
| `LPH-2025-TPL-0001` | Комплексная проверка контрагента | @username | 2025-11-25 | `prompts/kontragent/kontragent.md` |

---

**Как добавить новый fingerprint:**

1. Найди последний номер в нужной категории
2. Увеличь на 1
3. Добавь запись в таблицу
4. Укажи fingerprint в YAML-шапке промпта
